Server folder of the projec is the back-end part of the project. Here we are using MongoDB as our database
server. The launch the server we have to run the command `npm start`.

Here we are going to write the API requests from controllers and map them through the routes
to find their endpoints.
auth.js ⇒ /auth
hst_complaints.js ⇒ /hst
smesspage.js ⇒ /smess
supdates.js ⇒ /supd
whst_complaints.js ⇒ /whst
wupdates.js ⇒ /wupd
outpass.js => /out
woutpass.js => /wout

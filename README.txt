This is a readme file for the DBMS project Students Portal.

There are 2 halves of this project:
1. Client
2.Server

Client folder of the project is the front-end part of the project. This runs on a local server and has to be
initialized by the running the command `npm start`.

Server folder of the projec is the back-end part of the project. Here we are using MongoDB as our database
server. The launch the server we have to run the command `npm start`.
